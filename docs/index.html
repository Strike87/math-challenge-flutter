How to Play
Game Modes
  • Standard: Answer questions before time runs out
  • Blitz ⚡: 60 seconds — answer as many as possible
  • Death 💀: One wrong answer = Game Over!
  • Master 🏆: Story mode with boss battles
  • Survival 💪: 3 hearts, endless questions — difficulty rises every 5 correct. Boss appears every 10!
  • Combo 🔥: 90 seconds — build a streak for ×2, ×3, ×5 multipliers. One miss resets it!
Power-Ups
  • +5s: Adds 5 seconds to the current timer
  • 50/50: Removes 2 wrong answers
  • ×2: Doubles points for the next correct answer
  • 🛡️ Shield: Blocks one wrong answer
  • ⏸️ Freeze: Pauses the timer for the current question
  • 🔀 Swap: Instantly swaps to a new question — no penalty!
Keyboard Shortcuts
  • Keys 1 2 3 4 or arrow keys to select answers
  • Space to skip, Esc to quit
